# Columbia-Asia-Hospital-Analysis
This project analyzes hospital operations, revenue performance, and patient behavior using Power BI, SQL, and Excel.
The project includes interactive Power BI dashboards tailored for management, doctors, and patients, tracking KPIs such as revenue, wait time, satisfaction, and visit patterns.

Key analyses include department-wise revenue and growth trends using DAX, doctor performance and patient demographics using SQL, and operational insights to identify high-demand departments and staffing gaps. Data-driven discount and retention strategies were proposed to improve patient loyalty.

This project demonstrates how analytics can support better decision-making, operational efficiency, and improved healthcare outcomes.
